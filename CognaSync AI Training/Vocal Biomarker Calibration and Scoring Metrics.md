In the Mental Fitness Vocal Biomarker (MFVB) algorithm, the extracted vocal features were **normalized using distributions calculated from a large proprietary digital biobank** 1\. This reference database was compiled from data acquired during previous clinical studies conducted across multiple hospital sites in India 1\.  
By comparing and normalizing the raw vocal feature values against this India-based reference population, the algorithm is able to translate the data into a single composite score from 0 to 100 1\. This normalization process also helped establish the thresholds for the user-friendly score categories of "Excellent," "Good," and "Pay Attention" 1, 2\.
